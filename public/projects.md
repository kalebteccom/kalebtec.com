## Overview

Kalebtec's projects page showcases our portfolio of technology consulting work. Projects are managed through Payload CMS and are filterable by industry and technology.

Each project includes:

- **Title**: The name of the project
- **Client**: The company or organization the work was performed for
- **Description**: A summary of what was built or delivered
- **Industries**: The business sectors the project relates to (e.g., fintech, healthcare, SaaS)
- **Technologies**: The specific tools and frameworks used (e.g., Next.js, React, AWS, Python)
- **Featured image**: A visual representation of the project
- **Published date**: When the project was completed
- **Rich text content**: Full case-study details where available

## Filtering

Visitors can filter the project listing by:

- **Industry**: Filter by business sector to see relevant work
- **Technology**: Filter by specific technologies or frameworks

## Project Detail Pages

Each project has its own detail page at `/projects/[slug]` with:

- Full-resolution featured image
- Client and date information
- Industry and technology tags
- Complete project description
- Rich text case-study content
